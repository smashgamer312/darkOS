Я устроил систему драйверов! Как? Сейчас расскажу!
Итак. В папке system есть файлы .html с пометкой #d. Это и есть драйвера.
Драйвера помогают основному ядру выполнять некоторые функции. Они подключаются к файлу main.html для работы. Также в новом обновлении... А вот это уже позже)

Как запустить?

1. Скачайте пайтон (python)
2. Зайдите в дерикторию папки (не в system, там где main.html)
3. Прописываем python3 -m http.server 8000
4. Готово! Зайдите по ссылке http://localhost:8000/main.html
